# 集氣網頁修改操作說明（給不寫程式的同仁）

用 AI 工具（Codex）就能自己改集氣網頁的文字、圖片、課程、顏色，不用等工程師。

## 開始之前


1. 啟動 Codex：
   ```
   codex
   ```

## 第一句話一定要先講這個

Codex 不會自動知道這個專案的規則，**每次開新的對話**，第一句話請貼這段：

```
請先讀取 .claude/skills/power-cheer-frontend/SKILL.md 和
.claude/skills/power-cheer-frontend/reference.md 這兩份文件，
之後修改 power-cheer 專案都要照裡面的規則跟步驟做。
```

Codex 讀完之後，就會知道哪個東西要改哪個檔案、改完要怎麼檢查、怎麼上線。

## 接著就可以直接說需求

用平常講話的方式描述就好，例如：

- 「幫我把第 3 堂課的封面圖換成這個網址：xxx」
- 「活動說明文字改成：xxx」
- 「折扣門檻改成滿 150 人打 9 折」
- 「主色改成比較深一點的藍色」

不用講程式術語，也不用自己找檔案，Codex 會自己去找該改的地方。

## 改完之後

Codex 會告訴你這次改動要怎麼「上線」（有時是 `git push`，有時是 `clasp push`）。

- 前端（畫面）的改動：照 Codex 說的步驟做，或請工程師幫忙推上去。
- 後端（Google Apps Script）的改動：**推上去之後還要多一個步驟**才會真的生效，建議請工程師收尾確認，避免改了卻沒生效。

## 小提醒

- 不確定的地方直接問 Codex 就好，不用怕問。
- 想砍掉某個功能、改姓名/Email 的驗證規則、改 24 小時集氣限制這種比較大的改動，請先跟工程師討論一下再動手。
- 改完記得實際打開網頁看一下有沒有跟預期一樣。
